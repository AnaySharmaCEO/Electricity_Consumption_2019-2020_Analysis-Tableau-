Project Overview

Electricity consumption is a key indicator of economic activity. This project analyzes electricity usage across Indian states using data from POSOCO weekly energy reports covering the period from January 2019 to December 2020. Through data analysis and visualization, the project explores consumption trends, regional differences, and the effects of the COVID-19 lockdown on electricity demand.
